------------------------ README -----------------------------
Language Used: Pyhton 2.7.6

Instructions to run:
1. Install all the packages used.
2. Open Terminal.
3. cd to the folder where the file reduce.py is stored.
4. Before running the code delete the clusteredImages folder to create a new clusteredImages folder containing reduced image files.
5. Run the file using the command – ‘python reduce.py’.
